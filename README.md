# Calculator

This is a simple calculator application that allows you to perform basic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division

### Usage
1. Enter the first number.
2. Select the operation.
3. Enter the second number.
4. Click on the "=" button to see the result.
